# 海巡署考試題庫

完整的海巡署考試題目練習系統，提供即時反饋、進度追蹤和統計分析功能。

## 功能特色

### 📖 官方題庫
- 完整收錄 119 道真實海巡署考試題目
- 涵蓋法律、海事法規和實務案例分析
- 題目採用繁體中文

### 🏆 即時反饋系統
- 選擇答案後立即顯示視覺化回饋
- 答對時以綠色和 ✓ 標示
- 答錯時以紅色和 ✗ 標示，並顯示正確答案
- 其他選項在選擇後會變暗

### 📊 進度追蹤
- 使用瀏覽器本地儲存保存進度
- 自動從上次作答位置繼續
- 即時顯示完成百分比
- 題目計數器（例如：「第 5 題 / 共 119 題」）

### 📈 統計儀表板
- 已完成題數統計
- 答對/答錯題數統計
- 正確率百分比
- 視覺化進度條
- 可重置所有進度

### ❌ 錯題回顧
- 複習所有答錯的題目
- 顯示你的答案與正確答案對比
- 幫助聚焦在弱點領域

### 📱 響應式設計
- 手機優先的設計方式
- 針對手機、平板和桌機優化
- 現代化清爽介面與流暢動畫
- 觸控友善的操作界面

## 技術架構

- **HTML5**: 語意化標記
- **CSS3**: Flexbox/Grid 佈局、CSS 變數、動畫效果
- **原生 JavaScript (ES6+)**: 模組化程式碼組織
- **LocalStorage API**: 進度持久化
- **無外部依賴**: 純 HTML/CSS/JS 實作

## 使用方式

### 本地執行

1. **複製專案**:
   ```bash
   git clone https://github.com/yaya9689/question-bank-test.git
   cd question-bank-test
   ```

2. **啟動本地伺服器**:
   ```bash
   # 使用 Python 3
   python3 -m http.server 8080
   
   # 或使用 Node.js
   npx http-server -p 8080
   ```

3. **在瀏覽器開啟**:
   瀏覽到 `http://localhost:8080`

### 部署

本應用程式可部署到：
- **GitHub Pages**: 在專案設定中啟用 GitHub Pages 即可
- **Netlify/Vercel**: 零配置部署
- **任何靜態網站主機**: 上傳所有檔案到你的主機供應商

不需要建置流程或伺服器端程式碼！

## 如何使用

1. **開始**: 點擊首頁的「開始練習」按鈕
2. **作答**: 點擊任一選項（A、B、C 或 D）選擇你的答案
3. **反饋**: 立即顯示視覺化回饋，告訴你答對或答錯
4. **下一題**: 點擊「下一題 →」按鈕繼續
5. **追蹤**: 隨時透過「查看統計」查看你的統計資料
6. **複習**: 透過「錯題回顧」檢視答錯的題目

## 題目格式

題目儲存於 `data/questions.json`：

```json
{
  "id": 1,
  "question": "海巡署之主要任務為何？",
  "options": {
    "A": "海域執法",
    "B": "海洋保育",
    "C": "海岸巡防",
    "D": "以上皆是"
  },
  "correctAnswer": "D"
}
```
